# IndiaMART Acronyms — Quick Reference

| Acronym | Full Name | Context |
|---------|-----------|---------|
| BL | BuyLead | Lead from buyer to seller |
| MCAT | Microcatalog | Product categorization system |
| ISQ | Item Searchable Quantity | Seller product quality metric |
| NI | Not Interested | Buyer feedback on a BuyLead |
| PNS | Phone Number Service | Caller/callee service for buyer-seller calls |
| GLID | Global Login ID | Unique seller identifier |
| TOV | Transaction Order Value | Seller order value metric |
| FCP | Free Customer Profile | Non-paying seller account |
| CTA | Call To Action | UI button/link element |
| PDP | Product Detail Page | Product listing page on IndiaMART |
| DLP | Domain Landing Page | Category landing page |
| HRS | High Risk Seller | Flagged seller account |
| AutoBL | Auto BuyLead | System-generated buy leads |
| GLA | GL Admin | Internal admin tool |
| VANI | Voice AI Agent | AI voice bot for seller calls |
| CaaS | Component as a Service | Frontend micro-service architecture |
| LMS | Lead Management System | Seller tool for managing BuyLeads |
| PLA | Price List Aggregate | Pricing data aggregation |
| BBL | Below BlueLine | Low-quality seller tier |
| BLNI | BuyLead Not Interested | Buyer rejection of a lead |
